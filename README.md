# Arbiter API

A real time application to source order books from multiple exchanges

### Running the application

First setup the env

```
cp .env.sample .env
```

Install dependencies

```
npm install
```

Start the application

```
npm start
```

## Running the tests

Runs all *.spec.js tests and reports coverage via istanbul

```
npm test
```
